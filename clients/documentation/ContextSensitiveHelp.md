# Context-Sensitive Help

This document mirrors the in-app help panels and tooltips.

## Dashboard
- Review key stats at a glance.
- Click a recent task to open details.
- Use New Task to create work fast.

## Task Board
- Use toggles to show or hide columns.
- Edit or delete the current project.
- Click a row to view task details.

## All Tasks
- Select rows to run bulk actions.
- Search to find tasks quickly.
- Click a row to open details.

## Calendar
- Create tasks from the calendar view.
- Click a task chip to open details.
- Use arrows to switch months.

## Timeline
- Only tasks with dates appear here.
- Click a bar to open details.
- Use New Task to add dated work.

## My Tasks
- Cards show tasks assigned to you.
- Click a card to open details.
- Create a new task from the header.

## Analytics
- Track completion and overdue totals.
- Compare status and priority distribution.
- Use project selection to focus metrics.

## Workflows
- Select a workflow to see details.
- Toggle status to activate or pause.
- Create a workflow to automate actions.

## Account
- Update profile, password, and email.
- Review sessions and revoke access.
- Set language and theme preferences.

## Task Detail
- Edit or delete from the header.
- Review dates, priority, and assignees.
- Add comments for context.

## Task Form
- Pick a project and enter a clear title.
- Set status, priority, and dates.
- Assign users or groups before saving.

## Auth
- Enter tenant and credentials.
- Use links for signup or reset.
- Check status messages after submit.
